# 112indewijk

Mobielvriendelijk buurtplatform voor meldingen en veiligheidsinformatie.

## Opzet

De site bestaat uit gewone HTML, CSS en JavaScript:

- `index.html` - pagina-opbouw, navigatie, hero, meldingen, buurtbeeld en informatieblokken
- `styles.css` - volledige responsive styling voor desktop en mobiel
- `app.js` - voorbeeldmeldingen, filters, detailpaneel, buurtstatistieken en mobiel menu

## Belangrijk

Deze versie gebruikt voorbeelddata. Er is nog geen live P2000-feed, database, backend, kaart of login gekoppeld.

## Lokaal bekijken

Open `index.html` direct in de browser of start een lokale server:

```bash
python -m http.server 3000
```

Daarna openen:

```text
http://localhost:3000
```
